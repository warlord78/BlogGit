Hola esta sera la primera linea de Comandos basicos que puedes aplicar en Git y trabajar de una manera los proyectos mas rapidos Directorio de Trabajo / Staging / Repositorio Local

    Comandos Basicos

  • Git Init (Inicio del Repositorio)
  • Git Add . (Add Archivos)
  • Git Commit -m "Primera version del archivo".

Git es una herramineta muy poderosa que nos sirve para que un grupo de personas trabajen el mismo archivo, le hagan cambios completamente nuevos y luego fusionarlos en una unica version y asi trabajar mas ordenado posible un proyecto grande.
